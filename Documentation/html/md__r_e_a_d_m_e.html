Introduction

This is a simple .Net Core library provided by Panviva Pty. Ltd. to help use Panviva APIs reliably and easily in a .Net Core developer environment.

This is the Base Repository of Panviva.Sdk which includes

  • Panviva.Sdk.Models
  • Panviva.Sdk.Services.Core

Setup

  1. Import necessary using statements.

    ```c# 

    using  Panviva.Sdk.Models.V3.Get;

using  Panviva.Sdk.Services.Core.Extensions;

using  Panviva.Sdk.Services.Core.Exceptions

    ```

  2. Add cofigs to appsettings.

    ```json 

    "PanvivaApi": {

"BaseUrl": "--Base Url--",

"ApiVersion": "--Version--",

"Instance": "--Instance--",

"ApiKey": "--Api key--",

"RetryCount": "--RetryCount--"

}

    ```

Note: RetryCount is optional, If not specified the libry will use the default retry count of 3

  1. Add middleware and Service.

    • To Configure method

    ```c# 

    app.UsePanvivaExceptionMiddleware();

    ```

    • To ConfigureServices method

    ```c# 

    services.AddPanvivaApis();

    ```

  2. Inject the handlers you want to use.

    ```c# 

    public MyClassConstructor(IQueryHandler queryHandler, ICommandHandler commandHandler)
{
    this.queryHandler = queryHandler;
    this.commandHandler = commandHandler;
}

    ```

  3. Using Get requests with QueryHandler

    • First add necessary usings.

    ```c# 

    using Panviva.Sdk.Services.Core.Handlers.V3;
using Panviva.Sdk.Services.Core.Domain.QueryModels.V3;

    ```

    • Use Query Handler to Get Api results ```c# 
      GetDocumentQueryModel myQueryModel = new GetDocumentQueryModel 
{
    id = "123";
} 

GetDocumetResultModel myResultModel = await this.queryhandler.HandleAsync(myQueryModel);

    ```

  4. Using Post requests with CommandHandler

    • First add necessary usings.

    ```c# 

    using Panviva.Sdk.Services.Core.Handlers.V3;
using Panviva.Sdk.Services.Core.Domain.CommandModels.V3;

    ```

    • Use Query Handler to Get Api results ```c# 
      LiveSearchCommandModel myCommandModel = new LiveSearchCommandModel 
{
    UserName = "smapleName",
    Query = "SampleQuery",
    MaximizeClient = true,
    ShowFirstResult = true,
} 

LiveSearchResultModel isSuccess = await this.commandHandler.HandleAsync(myCommandModel);

    ```
